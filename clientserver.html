Client/Server Computing

By: Ahmad Fadel

What is the client/server model?

The client/server model acts as a core network which enables building functionality for web technologies. These Clients include Web browsers, chat applications, and email software, among others. Servers include Web, database, application, chat and email, etc. Clients request specific processes and data and the server relays process output to the client

Advantages of the client/server model

Several major advantages of the client/server model are:

  1. Security:Password management provides a security measurement for this by requiring a minimum amount of characters to be used in passwords checking passwords for guess-ability, and regularly asking users to change their passwords
  2. Accessibility:You can access many different servers and databases
  3. Centralization:It's used with general-purpose computer systems, single user system, and multi user system
  4. Proper Management:The communication between clients and servers is a two way street. Servers can contact clients to ensure that the client has appropriate patches, updates, or to see if the client needs anything else

Examples of TCP/IP protocols

  • FTP-FileZilla,WinSCP,Cyberduck
  • HTTP-is an application protocol for distributed, collaborative, and hypermedia information systems
  • HTTPS-is the secure version of HTTP
  • DHCP-is a client/server protocol that automatically provides an Internet Protocol (IP) host with its IP address and other related configuration information such as the subnet mask and default gateway
  • SMPT-Yahoo,Hotmail,Outlook,Gmail

Examples of Network Utilities and Services

  • Ping-Tests connectivity between two hosts
  • Tracert-Identifies pathways taken along each hop, rather than the time it takes for each packet to return
  • Pathping-Provides useful information about network latency and network loss at intermediate hops between a source address and a destination address
  • Whois -Is a query and response protocol that is widely used for querying databases that store the registered users or assignees of an Internet resource
  • IPConfig-Command line utility will show detailed information about the network you are connected too
